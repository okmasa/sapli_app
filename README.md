# サプリアプリ
サプリメントのレビューサイトです。

「サプリメントの評価」と「感想の投稿」を共有することができます。

URL: https://www.sapliapli.com

[![ホーム画像](app/assets/images/home-image.png)](https://www.sapliapli.com)

# 構成
- インフラ
  - AWS ( EC2 / ELB / Route53 / RDS / VPC )

- フロントエンド
  - HTML
  - CSS
  - JavaScript
  - jQuery

- バックエンド
  - Ruby 2.6.3
  - Rails 6.0.3

- Webサーバ
  - Nginx

- APサーバ
  - Unicorn

- データベース
  - RDS for MySQL 5.6.1
